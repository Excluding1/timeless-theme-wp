## 1. Customer-facing summary

**Description:**
> We replace all silicone joints in your shower (corners, floor-wall, wall-wall, screen seal). Removes mould-affected old silicone; installs new mould-resistant Selleys Wet Area silicone for fresh, watertight finish. 24hr cure.

**T2 scope:**
- All silicone joints in shower enclosure removed
- Mould treatment of substrate before re-silicone
- New Selleys Wet Area silicone (mould-resistant, white standard)
- Crisp masking lines for clean appearance

**NOT included:**
- Bath silicone (separate SKU SIL-02)
- Grout regrout (RSC series)
- Tile/screen replacement

## 6. Step-by-step recipe

### Phase 1: Arrival + assessment (5min)
1. SMS sent if not riding on bigger job
2. 2 photos before: shower interior + corner detail
3. **STOP triggers:**
   - Mould visible BEHIND tile (water ingress) → office referral
   - Silicone holds water (substrate already wet) → must dry first; reschedule

### Phase 2: Setup (15min)
1. Drop sheet outside shower
2. Mask: shower screen frame, tap fixtures, drain (anywhere overspray could land)
3. Don gloves

### Phase 3: Old silicone removal (25min)
1. Cut along old silicone with utility knife (fresh blade)
2. Pull out in continuous strip — best practice = no fragments
3. If brittle/mouldy: scrape with blade, vacuum residue
4. Wipe joint with methylated spirits to remove silicone oils

### Phase 4: Mould treatment + clean (10min)
1. Apply Exit Mould or similar to any visible mould; dwell 5min; wipe
2. Final clean with meths
3. Joint must be uniformly dry + clean before silicone

### Phase 5: New silicone bead (30min)
1. Apply 3M Blue tape parallel to joint (2mm gap, both sides) — defines clean lines
2. Cut Selleys nozzle at 45° angle, 5mm aperture
3. Run silicone bead in continuous single pass — DON'T pause mid-corner
4. Tool with wet finger or smoothing tool — single pass, no rework
5. Remove tape immediately while silicone wet (clean lines)

### Phase 6: Cleanup + handover (10min)
1. 1 photo during (post-removal, pre-bead)
2. 3 photos after: 3 corners showing crisp silicone
3. Pack tools, vacuum any debris
4. Leave aftercare card variant (silicone-only quick variant)
5. Walk customer through: 24hr cure no water, mould-resistant cleaning routine
6. Photo upload to SM8 + complete
