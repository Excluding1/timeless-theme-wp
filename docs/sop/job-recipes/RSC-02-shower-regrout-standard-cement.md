## 1. Customer-facing summary

**Description:**
> We remove all old grout from your shower tiles, replace with fresh cement grout in your colour choice, and reseal all silicone joints (corners, floor-wall, wall-wall). Restores waterproofing + appearance. 24hr cure before use.

**T2 Recommended scope:**
- Full grout removal (all wall + floor tile lines in shower enclosure)
- Premium cement grout install (Davco Sanitised Coloured)
- All silicone joints replaced (Selleys Wet Area, mould-resistant)
- Mould treatment + deep clean of tile surfaces
- 2-year warranty on grout, 1-year on silicone

**NOT included:**
- Tile replacement (TRP series)
- Bath/basin resurface (BTH/BSN series)
- Floor tile regrout outside shower (FBR series)
- Epoxy upgrade (RSE-02 = +$200 sub + $80 materials)

## 6. Step-by-step recipe

### Phase 1: Arrival + assessment (15min)
1. "On my way 30min" SMS sent
2. Greet, confirm scope (shower regrout + silicone, T2 standard cement)
3. 3 photos: wide shower interior + close-up old grout + corner where old silicone visible
4. **STOP triggers:**
   - Pre-1990 + tile damage → asbestos backer board concern
   - Tiles loose / falling → tile fix needed first (different SKU TRP-01)
   - Mould smell deep / black mould visible behind tile → contractor referral

### Phase 2: Setup + mask (30min)
1. Drop sheet on shower floor outside enclosure
2. Mask: shower screen seal, tap fixtures, drain
3. Open window / fan
4. Don gloves + dust mask

### Phase 3: Old grout removal (90min)
1. Oscillating multi-tool with grout-removal blade
2. Work systematically: top tile rows down, then floor
3. Depth: remove minimum 3mm grout (clean back to clean substrate)
4. Vacuum debris between rows (don't let grit accumulate)

### Phase 4: Clean + mould treatment (40min)
1. Vacuum all tile lines thoroughly
2. Wipe with methylated spirits (degrease)
3. Apply Exit Mould to any visible mould; dwell 5min; wipe
4. Final wipe — surface should be uniformly dull + clean

### Phase 5: New cement grout install (90min)
1. Mix Davco Sanitised Coloured Grout per packet ratio
2. Apply with grout float at 45° angle, working diagonally across tile lines
3. Press grout firmly into all gaps; remove excess immediately
4. Wait 15-20min (per packet — partial set)
5. Wipe excess with damp sponge, working in circular motion (not along lines)
6. Wait full set time (30min usually)
7. Polish off haze with dry microfibre

### Phase 6: Old silicone removal + new bead (75min)
1. Cut along old silicone with utility knife (fresh blade per corner)
2. Pull out in continuous strip (best practice — no fragments left)
3. Clean joint with methylated spirits
4. Apply 3M Blue tape parallel to joint (2mm gap, both sides)
5. Run new Selleys silicone bead — single continuous pass
6. Tool with wet finger or smoothing tool — single pass, no rework
7. Remove tape immediately while silicone wet (clean lines)

### Phase 7: Photos + handover (25min)
1. 4 final photos: full shower, corner detail, floor-wall, wall-wall
2. Leave Card 1 (Regrout aftercare) on vanity
3. Walk customer through: 24hr cure (no shower), gentle cleaners only, run fan after showers
4. Photo upload to ServiceM8
5. Mark complete

## 7. Quality criteria

| Criterion | Good | Fail |
|---|---|---|
| Grout consistency | Uniform colour, level with tile face | Patchy colour, low spots |
| Silicone bead | Uniform width, smooth single pass | Lumpy, multiple passes visible |
| Surface prep | All old grout removed to depth | Old grout visible at depth |
| Cleanup | Spotless tile face | Grout haze, silicone smears |
| Mould treatment | No visible mould | Black spots remaining |

**Photos:** 3 before (wide + close-up old grout + corner) + 2 during (post-removal + post-grout) + 4 after (4 angles).
